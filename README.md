# AI Workflow Audit (FL-01)

## Overview

This repository contains my **FL-01: Setup Phase – AI Workflow Audit** submission for the AI Fluency course.

The purpose of this project is to evaluate my weekly workflow, identify where Artificial Intelligence can be used effectively, configure essential AI tools, and establish a foundation for future AI-assisted work.

---

## Objectives

- Audit recurring weekly tasks
- Classify tasks based on appropriate AI usage
- Configure AI tools (ChatGPT, Claude, Anthropic Academy)
- Create a Claude Project with custom instructions
- Define three reusable AI-assisted tasks
- Document evidence of tool configuration
