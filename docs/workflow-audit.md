# Workflow Audit

## Overview

Before integrating AI into my daily workflow, I evaluated the recurring tasks I complete each week as a first-year BSc Information Technology student. The purpose of this audit is to determine which tasks should remain entirely my responsibility, which tasks benefit from AI collaboration, and which tasks can be delegated or automated while maintaining academic integrity.

## Weekly Workflow Audit

| Task | Classification | Rationale |
|------|----------------|-----------|
| Complete Python programming assignments | Collaborate with AI | AI helps explain concepts, debug code, and suggest improvements, but I must understand and write the final solution. |
| Build HTML, CSS, and JavaScript web projects | Collaborate with AI | AI assists with layout ideas and debugging while I implement and customize the project. |
| Write GitHub README documentation | Delegate to AI with review | AI drafts professional documentation quickly, but I review it to ensure accuracy. |
| Debug programming errors in VS Code | Collaborate with AI | AI identifies possible causes of errors while I test and verify the final solution. |
| Study networking concepts (OSI Model, TCP/IP) | Collaborate with AI | AI explains difficult concepts and creates practice questions, but I need to master the content myself. |
| Research assignment topics | Delegate to AI with review | AI summarizes information efficiently, but I verify sources and ensure the information is accurate. |
| Prepare for tests and examinations | Collaborate with AI | AI generates quizzes, flashcards, and explanations while I complete the studying and revision. |
| Organise my weekly study schedule | Fully automate | AI can generate a study timetable based on deadlines and priorities with minimal manual adjustments. |
| Check grammar and spelling in reports | Delegate to AI with review | AI improves clarity and grammar, but I ensure the final submission reflects my own work. |
| Create project planning checklists | Fully automate | AI efficiently converts project requirements into actionable task lists. |
| Push code to GitHub using Git | Just Me | Using Git confidently is an essential software development skill that I need to perform independently. |
| Write personal reflections and academic integrity declarations | Just Me | These must represent my own experiences, learning, and opinions and should not be generated by AI. |

## Summary

This audit shows that AI is most valuable as a collaborative assistant for learning, coding, and documentation. However, I recognise that some tasks require my own judgment, responsibility, and technical ability. Understanding these boundaries will help me use AI ethically and effectively throughout the remaining phases of the course.
